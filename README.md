## Description

Built using [Nest](https://github.com/nestjs/nest), [TypeORM](https://typeorm.io/#/), [Swagger](https://swagger.io/), [Compodoc](https://compodoc.app/), [MySql](https://www.mysql.com/), [Jest](https://jestjs.io/), [Typescript](https://www.typescriptlang.org/), [Prettier](https://prettier.io/), [ES Lint](https://eslint.org/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run server

# production mode
$ npm run start:prod

# run client 
$ npm run client


# code documentation mode
$ npm run doc:server
```


## Lint
```bash
# run lint for server
$ npm run lint:server

# run lint for server
$ npm run lint:client
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
